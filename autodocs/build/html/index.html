Main App¶

class wallymart.wallymart_app.WallymartApp(args=None, logger=None)¶

The WallymartApp object controls the execution of the program. As the central object or controller, the WallymartApp instantiates the database and the logger and is responsible for storing the wallymart.orm.credentials.Credentials and the wallymart.utils.shopping_cart.ShoppingCart.

It calls upon the wallymart.site.pages.Pages class, which controls subprograms within each block. In the first phase, the user is prompted to choose whether to create an account or log in as a customer or employee. After authenticating, if the user is a wallymart.orm.customer.Customer, he will be prompted to navigate through the wallymart.site.portal.customer_portal.CustomerPortal pages. Otherwise, if the user is an :class:Employee, he will be prompted to navigate through the wallymart.site.portal.employee_portal.EmployeePortal pages.

Variables

  • _logger (logging.Logger) – Stores an instance of the Logger class. This is used throughout the execution of the program.

  • _customer_or_employeer (str) – Stores ‘1’ for customer, ‘2’ for employee. This controls which loop the user enters after logging in.

  • _credentials (Credentials) – Stores an instance of the wallymart.orm.credentials.Credentials after the user authenticates.

  • _authenticated (bool) – After authenticating is set to true. Logging out will set this to False, which exits the program loops.

  • _shopping_cart – Stores an instance of the ShoppingCart

run()¶

Execute the program.

For troubleshooting, use the following to skip to the desired location:
Be sure to set the credentials first
Skip login for customer:
>>> self._customer_or_employee = ‘1’
>>> self._authenticated = True # use for testing
>>> self._credentials = Credentials(‘harrison’, ‘password’)
>>> self._credentials.set_user_id(1)
Go directly to checkout as a customer:
>>> self._shopping_cart = ShoppingCart([OrderItem(1, 2), OrderItem(2, 3)])
>>> self._shopping_cart.set_customer_id(1)
Skip login for employee:
>>> self._customer_or_employee = ‘2’
>>> self._authenticated = True # use for testing
>>> self._credentials = Credentials(‘harrison’, ‘password’)
>>> self._credentials.set_user_id(1)
